[1. Bitwise Operations](https://edabit.com/challenge/vvuAkYEAArrZvmp6X)

```js 

function bitwiseAND(num1, num2) {
    return num1 & num2;
}

function bitwiseOR(num1, num2) {
    return num1 | num2;
}

function bitwiseXOR(num1, num2) {
    return num1 ^ num2;
}

```
